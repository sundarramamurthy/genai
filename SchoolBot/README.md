**SchoolBot** is an intelligent WhatsApp chatbot powered by Google Gen AI (Gemini 2.0 Flash) that helps parents and students find the best schools in their city. It uses Google Search to provide real-time information about schools, boards (CBSE/ICSE), and admissions.

## Features

*   **School Discovery:** Find schools by category (CBSE, ICSE, International, Govt/KV).
*   **Locality Search:** Search for top-rated schools in specific neighborhoods.
*   **Admission Info:** Get details on admission dates and approximate fee structures.
*   **Interactive Menu:** Easy-to-use numbered menu system on WhatsApp.

## Tech Stack

*   **AI:** Google Gemini 2.0 Flash (via Google Gen AI SDK / ADK)
*   **Backend:** Python, Flask
*   **Messaging:** Twilio API (WhatsApp)
*   **Search:** Google Search Tool (ADK)
*   **Tunneling:** Ngrok (for local development)

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sundarramamurthy/genai.git
    cd genai/SchoolBot
    ```

2.  **Create and activate a virtual environment:**
    ```bash
    ./start.sh
    ```
    (Or manually: `python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt`)

3.  **Set up Environment Variables:**
    Create a `.env` file in the `SchoolBot` directory:
    ```env
    GOOGLE_API_KEY=your_google_api_key
    TWILIO_ACCOUNT_SID=your_twilio_sid
    TWILIO_AUTH_TOKEN=your_twilio_auth_token
    NGROK_AUTHTOKEN=your_ngrok_token
    ```

## Usage

### One-Click Start (Recommended)
Run the helper script to start both the server and the tunnel:
```bash
./run.sh
```

### Manual Start
1.  **Start the Server:**
    ```bash
    venv/bin/python3 app.py
    ```
2.  **Start the Tunnel:**
    ```bash
    venv/bin/python3 start_tunnel.py
    ```

### Configuration
1.  Copy the Ngrok URL (e.g., `https://xyz.ngrok-free.app`) generated by the tunnel.
2.  Go to **Twilio Console > Messaging > Sandbox Settings**.
3.  Paste the URL into the "When a message comes in" field and append `/webhook` (e.g., `https://xyz.ngrok-free.app/webhook`).
4.  Save.

### Chat
Send **"Hi"** to your Twilio Sandbox number on WhatsApp to see the main menu!
